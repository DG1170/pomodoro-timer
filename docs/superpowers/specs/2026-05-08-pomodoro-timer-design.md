# 桌面番茄钟 — 设计文档

## 概述

基于 Electron + Alpine.js 的桌面番茄钟应用。单窗口，Tab 切换计时、任务、统计、设置四个模块。内置多套配色主题和时钟样式，本地 JSON 文件持久化存储，通知方式可配置。

## 技术栈

- **Electron** — 桌面容器、系统通知、文件读写
- **Alpine.js** — 响应式 UI，单 HTML 文件，无需打包工具
- **原生 CSS** — CSS Variables 实现主题系统，无需预处理器
- **JSON 文件** — 任务、统计、配置存于用户数据目录

## 架构

```
electron/
  main.js          — 主进程：窗口创建、IPC 处理、JSON 存储、通知、声音
  preload.js       — 通过 contextBridge 向渲染进程暴露 IPC 接口
  store.js         — JSON 文件读写（tasks.json、stats.json、settings.json）
  notifier.js      — 通知 + 声音播放调度
src/
  index.html       — 单页面，Alpine.js 应用
  style.css         — CSS Variables 主题 + 时钟样式 + 布局
  app.js            — Alpine.js 状态管理：计时、任务增删改查、统计、设置
  timer.js          — 计时引擎：倒计时逻辑、阶段切换（工作/短休/长休）
  audio/
    bell.mp3         — 默认提示音
```

## 模块设计

### 1. 计时模块（默认 Tab）

- 时钟展示区（4 种样式，见下文）
- 按钮：开始 / 暂停 / 跳过
- 当前阶段标签（工作中 / 短休 / 长休）
- 番茄进度点：🍅🍅⬜⬜⬜
- 若从任务页选中任务，计时页上方显示任务名称
- 倒计时结束 → 按设置触发通知 → 自动进入下一阶段

### 2. 任务模块

- 添加任务：文本输入框 + 添加按钮
- 任务列表项：名称、番茄数（已完成 / 预估）、选中标记
- 点击任务选中 —— 选中后在计时页关联显示
- 每次完成番茄自动为选中任务 +1
- 已完成任务：删除线，下沉或隐藏
- 数据持久化至 `tasks.json`

### 3. 统计模块

- 卡片区：今日 / 本周 / 总计番茄数
- 柱状图：本周每日番茄数
- 按任务统计：任务名 → 番茄数
- 数据来自 `stats.json`，每次完成番茄时更新

### 4. 设置模块

- **时长**：工作 25min（默认）、短休 5min、长休 15min、长休间隔 4 轮
- **主题**：6 套内置配色，可视化预览
- **时钟样式**：4 种时钟展示形态
- **通知**：复选框 — 系统通知 / 提示音 / 窗口置顶
- **数据**：导出 JSON / 重置全部数据

## 主题系统

通过 `<html>` 的 `data-theme` 属性控制，6 套内置配色。所有颜色通过 CSS Variables 定义 —— 切换主题仅改变颜色值，时钟样式保持不变。

| 主题 | 主色调 | 氛围 |
|------|--------|------|
| 温馨复古（默认） | `#c95a3d` | 暖米色调 |
| 极简白 | `#4A90D9` | 浅灰底 |
| 暗夜专注 | `#00d4aa` | 深蓝黑底 |
| 森林绿 | `#4CAF50` | 柔绿底 |
| 薰衣草 | `#7C3AED` | 淡紫底 |
| 日暮橙 | `#F59E0B` | 暖琥珀底 |

## 时钟样式

4 种内置时钟展示样式，与主题独立。时钟颜色自动跟随当前主题。

1. **圆环进度环** — SVG 环形进度条包裹居中倒计时数字
2. **大号数字** — 超大数字居中，极简风格
3. **横向进度条 + 数字** — 顶部进度条 + 下方数字
4. **模拟钟面** — 圆形钟面，指针式倒计时

## 数据存储

所有数据以 JSON 存入 Electron `userData` 目录：

```
{userData}/
  pomodoro/
    settings.json   — 主题、时钟样式、时长、通知偏好
    tasks.json       — 任务列表及番茄计数
    stats.json       — 每日番茄记录
```

## 通知系统

三种独立通知渠道，用户可自由组合启用：

- **系统通知**：Electron `Notification` API，Windows 原生通知
- **提示音**：通过 HTML5 Audio 播放 `bell.mp3`
- **窗口置顶**：倒计时结束后短暂 `mainWindow.setAlwaysOnTop(true)`

## 计时逻辑

```
工作 (25min) → 短休 (5min) → 工作 (25min) → 短休 (5min)
→ 工作 (25min) → 短休 (5min) → 工作 (25min) → 长休 (15min)
→ 循环
```

- 每完成 4 个工作阶段 → 长休代替短休
- 跳过按钮：直接进入下一阶段
- 暂停键冻结倒计时，恢复后继续

## 错误处理

- JSON 读取失败 → 回退到默认值，控制台警告
- JSON 写入失败 → 内联错误提示，下次保存时重试
- 通知权限被拒 → 静默禁用系统通知选项

## 不做范围

- 全局快捷键
